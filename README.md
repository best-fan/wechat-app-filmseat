# we-app-filmseat
高仿大麦在线选座 微信小程序
### 说明：
微信小程序高仿大麦小程序在线选座功能    数据为静态构造数据    目前支持座位排布：
``` 10*20 ~30*40```以内

### 目录结构：
- img —  项目截图
- pages — 页面文件
- utils — 工具类
